# Dota 2 Turbo — Pick & Bans (патч 7.41d)

Драфтер для Dota 2 Turbo в стиле игрового клиента: Captains Mode и Captains Draft,
все 127 героев патча 7.41d, оригинальные звуки анонсера и реплики героев при пике.

**Играть:** https://alexanderparkin.github.io/dota2_turbo_pick_bans/

## Возможности

- **Captains Mode** — полный пул, порядок банов/пиков как в патче 7.40+ (баны 3-2-2 / 4-1-2, пики 1-3-1), 8/15 с на действие, резерв 65 с.
- **Captains Draft** — случайный пул 28 героев (по 7 на атрибут), 3 бана и 5 пиков, банк 45 с.
- **Защита от бана** — перед драфтом каждая команда открыто отмечает 2 героев (золотой щит): банить нельзя, пикнуть может любая команда. В CD — выбор из сгенерированного пула.
- **Монетка** решает, кто ходит первым.
- **Онлайн на двоих**: один создаёт комнату и получает код, второй входит по коду. Каждый занимает свободную сторону (Radiant/Dire), вписывает название и жмёт «Готов». Пики и баны — только за свою сторону; тайминги судит создатель комнаты. Соединение — P2P (PeerJS), сервер не нужен.
- Быстрый поиск: во время драфта просто начните печатать имя героя.
- По окончании: PNG пиков в буфер обмена (тройной клик по «Сохранить» — полная картинка с банами в порядке драфта), текстовый результат.
- Тайм-аут как в игре: бан сгорает, пик выбирается случайно.

## Локальный запуск

Просто откройте `index.html` в браузере (нужен интернет: картинки — Steam CDN, звуки — Dota 2 Wiki).

## Структура

Один файл `index.html` — вся логика внутри. Движок драфта (`RULES`, `buildSeq`, состояние `S`)
изолирован от DOM; сетевой слой — `NET` + PeerJS.

Изображения и звуки принадлежат Valve (Dota 2).
